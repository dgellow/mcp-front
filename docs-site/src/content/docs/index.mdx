## What is MCP Front?

MCP Front is a secure proxy server that adds OAuth 2.1 authentication to Model Context Protocol (MCP) servers. It enables Claude.ai to safely access your company's internal resources while maintaining strict security boundaries.

## Architecture Overview

<div class="architecture-diagram">
  <img src="/mcp-front/architecture.svg" alt="MCP Front Architecture" />
</div>

MCP Front acts as a protective shield between Claude.ai and your internal MCP servers:

1. **Claude.ai** connects to MCP Front using OAuth 2.1
2. **MCP Front** validates the authentication and checks permissions
3. **Authorized requests** are proxied to the appropriate MCP servers
4. **MCP servers** can be stdio-based (Docker) or HTTP-based services

## Quick Start

```bash
# Using Docker
docker run -p 8080:8080 \
  -e GOOGLE_CLIENT_ID=your-client-id \
  -e GOOGLE_CLIENT_SECRET=your-secret \
  -e JWT_SECRET=your-jwt-secret \
  -v ./config.json:/config.json \
  ghcr.io/dgellow/mcp-front:latest

# Using Docker Compose
docker-compose up -d
```
